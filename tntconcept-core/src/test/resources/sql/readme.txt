En esta carpeta se encuentran los ficheros con datos de pruebas 
para cada una de las versiones de la aplicación.

La idea es generar la version, insertar los datos depruebas necesarios y meter
el script pertiente con todos los datos de pruebas aqui.

La version de la carpeta por tanto ha de coincidir con la versión de la tabla
Version de la aplicación.
